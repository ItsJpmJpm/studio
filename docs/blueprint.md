# **App Name**: MindFlow AI Assistant

## Core Features:

- AI-Powered Dialogue Engine: Utilize generative AI to understand user intent, maintain natural conversation flow, and generate empathetic, contextually relevant responses.
- Persistent Contextual Memory Tool: An AI tool that remembers and prioritizes past conversation history to provide continuity and personalization, preventing redundant information requests.
- Dynamic Ambiguity Resolution: An AI tool that analyzes short or ambiguous user inputs, leveraging immediate conversational context to accurately infer the user's meaning and respond appropriately.
- Rich Text Output Formatting: Automatically format AI-generated responses using Markdown (e.g., bolding, bullet points, line breaks) to enhance readability and ensure scannability.
- Intuitive Chat Interface: A clean, modern user interface enabling seamless text input and clear display of the assistant's responses.
- Internal Rule-Following Tool: An AI tool that adheres strictly to predefined operational guidelines and constraints (e.g., avoiding hallucination) when formulating responses.

## Style Guidelines:

- Primary color: Muted blue-purple (#333399) to convey intelligence, trust, and professionalism, suitable for a sophisticated assistant.
- Background color: A very subtle lavender-gray (#EFEFF6) provides a clean, unobtrusive backdrop, maintaining focus on content in a light theme.
- Accent color: A vibrant sky blue (#5CB3D7) for calls to action and important highlights, creating contrast and drawing attention where needed.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly readable qualities, ideal for clear conversational text.
